Movies API

Endpoint Description Method URL Body Response
Returns a JSON object of ALL movies GET /movies - JSON
Returns a JSON object of a single movie GET /movies/:Title - JSON
Returns a JSON object of all movies of a certain genre GET /movies/genre/:Genre - JSON
Returns the description of a genre GET /genre/:Genre - JSON
Returns a JSON object of a movie-director by name GET /directors/:Name - JSON
Returns a JSON object of one user GET /users/:Username - JSON
Creates a new user from a JSON POST /users JSON 201
Updates the information of a user PUT /users/:Username JSON 201
Adds a movie to a user's list of favorite movies PUT /users/:Username/movies/:_id - 201
Removes a movie from a user's list of favorite movies DELETE /users/:Username/movies/:_id - 201
Removes an existing user from the database DELETE /users/:Username - 201
